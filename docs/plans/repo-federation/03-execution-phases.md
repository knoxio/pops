## 0. Conventions for the executing agent

| Concern           | Rule                                                                                                                                                       |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Worktree          | `git -C /Users/joao/dev/personal/pops worktree add ../pops-wt/<TASK-ID> -b feat/<TASK-ID>-<slug> main`                                                     |
| Branch            | always off `main`; never push to `main`. PR + auto-merge per `05-cicd-deployment.md` §A.                                                                   |
| Local gate (TS)   | `pnpm install --frozen-lockfile && pnpm lint && pnpm format:check && pnpm lint:boundaries && pnpm typecheck && pnpm test` — must be green **before** push. |
| Local gate (Rust) | `cargo fmt --all --check && cargo clippy --all-targets --all-features -- -D warnings && cargo test` (in the cargo workspace dir).                          |
| Lockfile rule     | A task may rewrite `pnpm-lock.yaml` **only** if it is `parallel-group: solo`. Parallel tasks must not change resolution (see DEPENDENCY-DAG §c).           |
| Type safety       | No `as any`, no `as unknown as T`, no `eslint-disable`/`ts-ignore`. Fix root cause.                                                                        |
| Commits/PRs       | No Claude co-author / reference anywhere. Images publish to `ghcr.io/knoxio`.                                                                              |
| Rollback          | Every task: `git worktree remove` the branch; if merged, `git revert <merge-sha>` (mechanical moves revert cleanly).                                       |

`parallel-group` legend: `solo` = must run alone (mutates a serializing shared file);
`PG-x` = may run concurrently with other members of the same group in separate worktrees.
