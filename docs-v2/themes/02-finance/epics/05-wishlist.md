# Epic 05: Wishlist

> Theme: [Finance](../README.md)

## Scope

Build the wishlist — savings goals with target amounts and progress tracking. Items represent things the user wants to buy, with a target price and tracked progress toward that goal.

- **Unlocks:** Nothing directly — standalone feature

## Out of Scope

- Integration with inventory (linking purchased wishlist items to inventory)
